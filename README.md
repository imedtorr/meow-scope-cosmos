# 🌙 Meowroscope

> Космическая астрология для котиков — звёзды шепчут, хвосты виляют, мурчание обязательно.

Магический лендинг с гороскопами, колесом зодиака, совместимостью и **интерактивным мурчанием** на главной картинке. Нажми на космического котика — и пусть вселенная загудит от purr. ✨

## 🚀 Как запустить

Нужны **Node.js 18+** и **npm**.

```bash
# 1. Клонировать репозиторий
git clone git@github.com:imedtorr/meow-scope-cosmos.git
cd meow-scope-cosmos

# 2. Установить зависимости
npm install

# 3. Запустить dev-сервер
npm run dev
```

Открой в браузере: **http://localhost:8080**

### Другие команды

```bash
npm run build    # production-сборка
npm run preview  # просмотр собранной версии
npm run lint     # проверка кода
```

## 🎧 Мурчание

На главной странице нажми на **космического котика** — заиграет звук мурчания. Повторный клик — тишина. Знаки зодиака на колесе — только визуал, без звуков.
